## 上下文
需要通过 `expo-blur` 模拟 Liquid Glass 风格，并在 iOS 端随系统暗夜模式切换。

## 目标 / 非目标
- 目标：在 iOS 平台实现玻璃质感背景与磨砂效果。
- 目标：iOS 端根据系统深浅色自动切换主题。
- 非目标：非 iOS 平台的深色模式支持。

## 决策
- 视觉实现：使用 `expo-blur` + 半透明层作为全局背景。
- 主题切换：使用 React Native 颜色方案 API 监听系统主题，仅在 iOS 上生效。

## 备选方案
- 原生材质 API：Expo 环境不可用，放弃。
- 全平台暗色适配：非当前需求范围。

## 风险 / 权衡
- 过度模糊可能影响性能，需要控制层级数量与模糊强度。
